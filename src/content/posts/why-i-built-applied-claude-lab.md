Applied Claude Lab is an engineering-focused project that explores how to use Claude in real software development and test automation workflows.

This is not an AI hype blog.

## What This Site Covers

- AI + C# engineering workflows
- AI for test automation systems (OpenTAP/PTEM)
- Token optimization and cost control
- Claude CLI and Claude Code real-world experiments
- Practical prompt engineering for deterministic output

## Why Build This

Most AI content online is generic productivity tips. Engineers need practical, applied examples with real prompts, real costs, and real results.

Every post here comes from actual experiments — not theory.

## What to Expect

- Weekly deep technical posts (1,000–1,500 words)
- Short micro posts with prompt tricks and CLI tips
- Real cost breakdowns and token optimization data
- Honest results, including failures

Stay tuned.
